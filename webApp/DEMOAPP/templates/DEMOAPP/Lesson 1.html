All coders have to start somewhere, and it's always a good idea to make sure your software is working properly.

We should begin by creating a new python program. First, make sure you have the following:
-Python 3.9 or 3.10
-Pip installer
-VS Code
Using VS Code's terminal (Terminal>New Terminal), type "pip install python"

Make sure to make a filepath that you will remember! Once you have, create a python file, call it something like "Hello World.py" Once you have done this, type "print('Hello World');" in the program window above the terminal. Then click Run>Start Debugging.

If you were successful, you should have a message in your terminal that says "Hello World"World.py
Congratulations, you have just finished your first lesson!
